- Search for books using google books api
- Filter books (relevance/newest)
- View full book information

### Local launch
```bash
# Clone the repo
git clone git@github.com:slammyy/booklib-react.git

# Change directory
cd booklib-react/

# Install dependencies
npm install

# Run app
npm run dev
```
